Apa itu JSON ?

JSON adalah singkatan dari (JavaScript Object Notation) sebuah format dara yang digunakan untuk pertukran dan peyimpanan data.
JSON merupakan bagian (subset) dari Javascript. JSON bisa dibaca dengan berbagai macam bahasa pemrograman seperti C, C++, C#, 
Java, Javascript Perl, Python, dan banyak lagi.
Jika dibandingkan dengan XML, JSON memiliki karakteristik yang lebih sederhana dan mudah dibaca.
